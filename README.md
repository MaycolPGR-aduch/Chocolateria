# tienda_mascotas Prueba para Kibernum/Talana.

## Project setup, clonamos el proyecto

```
git clone https://gitlab.com/felipei.mancilla/tienda_mascotas.git
```

## Vamos a la carpeta del proyecto

```
cd tienda_mascotas

```

## Ejecutamos

```
npm install

```

### Ejecutar

```

npm run serve

```
